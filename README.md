Scaffold a new Next.js 16 app prewired for Based CMS.

## Usage

```bash
npx @based-cms/get-started my-app
```

## What it scaffolds

- Next.js 16 App Router project template
- `cms-client` integration and starter section definitions
- `.env` guidance for `BASED-CMS-SLUG` and `BASED-CMS-KEY`

## Options

- `--template <name>`: select template (default: `nextjs`)
- `--local`: point generated app dependency to a local `based-cms-client` checkout
